# cyart-soc-team
---
##  Project Overview
This repository contains the full documentation and evidence for the Week 2 SOC Analyst training. The training covered the end-to-end Incident Response (IR) lifecycle, from theoretical understanding to host-based threat containment.

##  Repository Structure
* **Week2 Theoretical-knowledge cyart.pdf:** Covers Alert Priorities, NIST IR Lifecycle, and Incident Classification.
* **Week2 Practical-knowledge cyart.pdf:** Documentation of Alert Management Sheets, Investigation Logs, and Threat Intelligence validation.
* **Week2 cyart.pdf (Final Report):** Capstone project featuring live network triage (`netstat`), forensic hashing (`CertUtil`), and threat containment via host-based Firewall rules.

##  Tools & Skills Demonstrated
* **Detection & Triage:** Identifying unauthorized sockets and analyzing suspicious IPs via VirusTotal.
* **Forensics:** Evidence preservation using SHA256 cryptographic hashing.
* **Containment:** Real-time IP blacklisting using Windows Defender Firewall.
* **Reporting:** Drafting Incident Tickets and Professional Escalation Emails.
